# Tutorial 03 – Socket Programming in Java

Module: Client–Server Architectures   

## Topics Covered
- TCP socket programming in Java
- String-based communication using BufferedReader & PrintWriter
- Byte-stream communication using InputStream & OutputStream
- Client–Server message exchange
- Blocking I/O behaviour

## Exercises

### Exercise 1: String-Based Communication
- Server.java
- Client.java
- Uses BufferedReader and PrintWriter

### Exercise 2: Byte-Stream Chat Application
- SimpleChatServer.java
- SimpleChatClient.java
- Uses InputStream, OutputStream, byte buffers

## Learning Outcomes
- Understand differences between character streams and byte streams
- Implement basic TCP client-server applications
